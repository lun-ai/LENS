# Summary of Prolog Predicates

## linear_path(A, B)
This predicate determines if there is a direct connection between two points A and B without any branches in between. It's true when:
- A and B are the same point
- A connects to another point C without branching, and there's a linear path from C to B

## partition(A, B, C)
This predicate divides a set of elements A into two groups (B and C) based on specific connectivity conditions:
- Group B contains all elements that are in the same circuit as A and have a linear path back to A
- Group C contains all elements that are in the same circuit as A but don't have a linear path back to A

## partition_sizes(A, B)
This predicate calculates the sizes of the two groups created when partitioning elements around point A:
- It divides the elements into two groups using the partition predicate
- It counts how many elements are in each group
- It returns these two numbers as a pair in B

## optimal_partition_sizes(A, B)
This predicate finds the best possible partition sizes by:
- Starting with an empty partition [0,0]
- Calculating partition sizes for each element in A
- Comparing different partition size pairs
- Selecting the partition that maximizes the minimum size of the two groups
- Storing this optimal partition size pair in B
