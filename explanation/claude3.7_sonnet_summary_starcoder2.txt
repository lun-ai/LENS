# Summary of Prolog Predicates

## isolated_behind(A, B)
This predicate determines if one gate is isolated behind another. It returns true if:
- Gates A and B are the same, or
- There is a path of connected gates between them where all gates in the path are isolated behind gate B.

## partition(List, Group1, Group2)
This predicate divides a list of gates into two groups based on isolation relationships:
- Group1 contains gates that are isolated behind a reference gate
- Group2 contains gates that are not isolated behind the reference gate
The predicate ensures that gates are properly categorized based on their circuit connections and isolation status.

## partition_sizes(List, Sizes)
This predicate calculates the sizes of the two groups created by the partition predicate:
- It first partitions the list into two groups
- Then it counts how many gates are in each group
- Returns these counts as a pair of numbers

## optimal_partition_sizes(List, OptimalSizes)
This predicate finds the best way to partition gates to achieve the most balanced groups:
- It tries different partition arrangements
- Compares the sizes of the resulting groups
- Returns the partition sizes where the smaller group is as large as possible

These predicates work together to analyze how gates in a circuit are connected and isolated from each other, and to find optimal ways to divide them into groups.
